**Fix in TaskList.tsx:**
```typescript

  setSelectedTask(task);  // Set task first
  open();                 // Then open drawer

```

**Possible cause:** CORS or invalid URL

**Check:**
- task.user_input is a valid URL
- Image server allows CORS
- URL is accessible
